get_geocode(string_addr)
get_geocode() takes string(string_addr) as input/parameter.
This function is google API dependent.
Function makes the passed string as google searchable string by formatting it and brining it to uniformamity.
Returns geocode for passed string in list format list([x coord of addr,y coord of addr]).
ERROR HANDLING TIP: if returned list is [0,0], then passed string_addr is not located and needed to be added manually
